## Question 3: HTML Attributes
1. **Essential Attributes**
   - `href` (for navigation links)  
   - `src` (for images and audio)  
   - `action` (for form submission)  

2. **Class vs. ID**
   - `class`: Used for styling groups of elements (e.g., artist name, tagline).  
   - `id`: Used for unique identifiers (e.g., linking to specific sections like `#contact`).  

3. **Improved User Experience**
   - `alt` attribute improved accessibility by describing images for screen readers.
